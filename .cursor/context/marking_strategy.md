# Outstanding Mark Strategy

Target grade: 80–100.

Priority areas by weighting:
- Testing and coverage
- Search quality
- Code quality
- Video presentation
- GenAI critical evaluation

The implementation should:
- resemble a miniature real-world search engine
- include advanced but explainable features
- prioritise maintainability and modularity
- include strong defensive programming
- include professional Git workflow
- include excellent documentation

Potential advanced features:
- TF-IDF ranking
- phrase search
- query suggestions
- benchmarking
- stop-word filtering
- stemming

Avoid:
- unnecessary complexity
- over-engineering
- hard-to-explain abstractions
