# COMP3011 Coursework 2 Context

This project is a miniature search engine implemented in Python.

The system:
- crawls quotes.toscrape.com
- extracts and tokenises text
- builds an inverted index
- stores the index
- supports command-line searching

The coursework is based on:
- web crawling
- parsing and tokenisation
- inverted indexing
- query processing
- ranking and retrieval

Target grade:
Outstanding (80–100).

The project should resemble a simplified real-world search engine while remaining understandable and explainable in a 5-minute video demonstration.

Priorities:
- modularity
- testing
- clean architecture
- professional Git workflow
- critical AI evaluation

Avoid:
- over-engineering
- unnecessary frameworks
- difficult-to-explain abstractions
